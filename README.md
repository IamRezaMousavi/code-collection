# Code Collection

Code Collection is my personal space for exploring, documenting, and building — where programming, system administration, and automation come together in one unified repository.
It’s part reference, part playground, and part toolbox for curious developers and engineers

---

## Repository Structure

This repository is a **living lab** (Cross-platform) for development, systems, and DevOps exploration.  
It contains categorized folders for different languages, platforms, and tools.

- `c-codes`: C/C++ project and snippets - focusing on performance

- `docker-dev`: Docker and containerization lab - focusing on building isolated environments.

- `micro-modules`: Tiny embedded modules and experiments.

- `python-programs`: Scripts and application written in Python.

- `system-station`: An admin lab - includes Linux, BSD, and Windows configuration, scripts, and tools.
