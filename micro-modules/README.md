## ARM

ARM is a family of RISC instruction set architectures for computer processors. [wikipedia](https://en.wikipedia.org/wiki/ARM_architecture_family)

---

## AVR

AVR microcontrollers are 8-bit RISC single-chip microcontrollers based on a modified Harvard architecture. [wikipedia](https://en.wikipedia.org/wiki/AVR_microcontrollers)

---

## Toolchain

A toolchain is a set of software development tools used to build and otherwise develop software. [wikipedia](https://en.wikipedia.org/wiki/Embedded_software)

- ARM

    ```sh
    sudo pacman -S arm-none-eabi-gcc arm-none-eabi-newlib openocd
    ```

- AVR

    ```sh
    sudo pacman -S avr-gcc avr-libc avrdude
    ```
